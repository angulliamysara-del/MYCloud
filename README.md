# MYCloud

**MYCloud** adalah aplikasi *Cloud Storage* personal berbasis web yang dirancang untuk memberikan solusi penyimpanan file yang aman, cepat, dan responsif. Aplikasi ini mengintegrasikan **Google Drive API** sebagai mesin penyimpanan utama dengan *backend* **Node.js/Express** dan database **MongoDB**, serta dibalut dengan antarmuka modern menggunakan konsep *Glassmorphism* yang mendukung akses optimal baik di perangkat desktop maupun *mobile*.

---

**MYCloud** is a web-based personal cloud storage application designed to provide a secure, fast, and responsive file management solution. By integrating **Google Drive API** as the core storage engine with a **Node.js/Express** backend and **MongoDB**, the application features a modern *Glassmorphism* interface that ensures an optimal user experience across both desktop and mobile devices.
