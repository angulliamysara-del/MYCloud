# MYCloud - Personal Cloud Storage

**MYCloud** adalah aplikasi Penyimpanan Awan (*Cloud Storage*) personal yang dirancang dengan antarmuka modern dan elegan menggunakan konsep **Glassmorphism**. Aplikasi ini memungkinkan Anda untuk mengelola file secara mandiri dengan keamanan dan kecepatan tinggi.

*   **File Management**: Mengunggah (*upload*), mengunduh (*download*), dan menghapus file dengan fitur *Drag & Drop*.
*   **Real-time Preview**: Melihat langsung foto atau dokumen tanpa harus mengunduhnya.
*   **Storage Monitoring**: Indikator kapasitas bar untuk memantau sisa ruang penyimpanan.
*   **Dynamic UI**: Notifikasi interaktif dan animasi halus untuk pengalaman pengguna yang premium.
